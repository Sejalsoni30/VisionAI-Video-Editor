# 🎬 VisionAI - AI Video Studio

VisionAI is a sleek, MERN-stack video editing dashboard featuring a modern dark UI and AI-driven processing simulations.

## 🚀 Key Features
- **Neural Lab:** Dynamic AI-driven processing animations.
- **Pro Editor:** Seamless workspace with multi-panel layout.
- **Cloud Assets:** Direct integration with Cloudinary for media storage.
- **Animations:** High-end motion design using Framer Motion.

## 🛠 Tech Stack
- **Frontend:** React, Tailwind CSS, Framer Motion, Redux.
- **Backend:** Node.js, Express, Cloudinary.

## ⚙️ Setup
1. `npm install` (Frontend & Backend)
2. Add `.env` (Cloudinary credentials)
3. `npm run dev`
